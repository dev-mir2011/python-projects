# `Expense Tracker`

A **Python mini-project** to track daily expenses using a **GUI and CSV storage**.  
Built to practice Python fundamentals and desktop application development.

---

## Tech Stack

- Python
- Tkinter (GUI)
- CSV (Data Storage)

---

## Features

- Add expenses (date, category, description, amount)
- View expenses in a table
- Persistent storage using CSV
- Simple and user-friendly interface

---

## How to Run

```bash
$ python expense_tracker.py
```

## Learning Outcomes

- Tkinter widgets and layouts
- File handling with CSV
- Input validation
- GUI application structure

## Future Improvements

- Monthly expense summaries
- Category-wise analytics
- Charts using Matplotlib
- SQLite database integration
