# URL Shortener

A **full-stack URL shortening service** built with a Flask backend and a React frontend.  
This project helps understand **API-based architecture and frontend–backend communication**.

---

## Project Structure

```
URL-Shorterner/
├── Flask-Backend/
├──  url-shortener-frontend/
|    ├── public/
|    └── src/
|    ├── assets/
|    └── components/
|    └── views/
```

---

## Tech Stack

- **Backend:** Python, Flask
- **Frontend:** React
- **Architecture:** REST API

---

## Features

- Generate short URLs
- Redirect to original URLs
- Frontend and backend separation
- Clean project organization

---

## Learning Outcomes

- REST API development
- URL routing and redirection
- Full-stack development workflow
- React + Flask integration
