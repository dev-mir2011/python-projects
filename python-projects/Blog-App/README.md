# `Blog App`

A **full-stack blog application** with a Python backend and a modern React frontend.  
This project focuses on **separating backend and frontend concerns** while building a real-world web application.

---

## Project Structure

```
Blog-App/
├── Backend/
├── Frontend/
|    └── blog-web-app/
|    ├── public/
|    └── src/
|    ├── assets/
|    │ ├── feature-images/
|    │ ├── json-image/
|    │ └── step-images/
|    └── components/
|    ├── elements/
|    └── views/
```

---

## Tech Stack

- **Backend:** Python, Flask
- **Frontend:** React
- **Styling:** CSS
- **Architecture:** Component-based frontend

---

## Features

- Blog post rendering
- Modular React components
- Structured asset management
- Clear backend–frontend separation
