# Weather App

A **desktop weather application** built using **Python and Tkinter**.  
The app fetches and displays **real-time weather data** for a given location.

---

## Tech Stack

- Python
- Tkinter (GUI)
- Weather API (OpenWeatherMap or similar)
- Requests library

---

## Features

- Search weather by city name
- Displays temperature and conditions
- Simple and clean GUI
- API-based data fetching

---

## How to Run

1. Install dependencies:

```bash
pip install requests
```

2. Run The app:

```bash
python weather-app.py
```

## API Key Setup

1. Get an API key from OpenWeatherMap

2. Add the API key inside the source code or as an environment variable
