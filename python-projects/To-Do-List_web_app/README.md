# To-Do List Web App

A simple **task management web application** built using Python and Flask.  
This project focuses on **CRUD operations and server-side rendering**.

---

## Project Structure

```
To-Do-List_web_app/
├── static/
└── templates/
```

---

## Tech Stack

- Python
- Flask
- HTML
- CSS

---

## Features

- Add new tasks
- View task list
- Simple and clean UI
- Template-based rendering
