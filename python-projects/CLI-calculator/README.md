# CLI Calculator

A simple **command-line calculator** built using Python.  
This project focuses on **basic programming logic, user input handling, and control flow**.

---

## Tech Stack

- Python
- Command Line Interface (CLI)

---

## Features

- Addition
- Subtraction
- Multiplication
- Division

---

## How to Run

```bash
python CLI-calculator.py
```

---

## Learning Outcomes

- Handling user input from CLI
- Conditional statements
- Functions
- Error handling (e.g., division by zero)
