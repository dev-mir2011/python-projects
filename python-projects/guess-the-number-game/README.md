# Guess the Number Game

A fun **Guess the Number** game built using **Python and Tkinter**.  
The player guesses a randomly generated number, and the app provides feedback.

---

## Tech Stack

- Python
- Tkinter (GUI)
- Random module

---

## Features

- Random number generation
- User-friendly GUI
- Feedback for high / low guesses
- Restartable gameplay

---

## How to Run

```bash
python guess-the-number-game-GUI.py
```

## Learning Outcomes

- Tkinter window and widgets
- Event-driven programming
- Random number generation
- Basic game logic
